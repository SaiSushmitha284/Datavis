Binary Search Tree (BST)

A Binary Search Tree (BST) is a data structure that stores elements in a sorted way to allow fast lookup, insertion, and deletion. Each node follows the rule: 

    1. The left subtree of a node contains only nodes with keys less than the node's key.
    2. The right subtree contains only nodes with keys greater than the node's key.
    3. Both the left and right subtrees must also be binary search trees.

Applications of Binary Search Tree:

 
    - Implementing search trees in databases.
    - Creating associative arrays or sets.
    - Efficient search, insertion, and deletion operations.
    - Used in expression parsing.

How BST Works?

A BST works by recursively dividing the elements. When searching for a value, the algorithm compares the value to the root and decides whether to move left or right.

Operations like searching, insertion, and deletion take time proportional to the height of the tree.

// Define a Node structure
typedef struct Node {
    int data;
    struct Node* left;
    struct Node* right;
} Node;

// Create a new node
Node* createNode(int data) {
    Node* newNode = (Node*)malloc(sizeof(Node));
    newNode->data = data;
    newNode->left = newNode->right = NULL;
    return newNode;
}

// Insert a node into the BST
Node* insert(Node* root, int data) {
    if (root == NULL) {
        return createNode(data);
    }
    if (data < root->data) {
        root->left = insert(root->left, data);
    } else if (data > root->data) {
        root->right = insert(root->right, data);
    }
    return root;
}

// In-order traversal of the BST
void inOrder(Node* root) {
    if (root != NULL) {
        inOrder(root->left);
        printf("%d ", root->data);
        inOrder(root->right);
    }
}

// Main function to test the BST
int main() {
    Node* root = NULL;
    root = insert(root, 50);
    insert(root, 30);
    insert(root, 70);
    insert(root, 20);
    insert(root, 40);
    insert(root, 60);
    insert(root, 80);

    printf("In-order traversal of the Binary Search Tree:\n");
    inOrder(root);

    return 0;
}
